# Aclo Kids WebApp

An informative web application with E-Commerce features built for **Aclo Kids**, built with the **MERN stack** (MongoDB, Express, ReactTS, Node.js) and styled using **TailwindCSS**.

## üöÄ Features
- **Educational Content** - Comprehensive educational information and engaging infographics on learning towers, including developmental benefits and best-practice usage guidance for parents.
- **E-Commerce Functionality** - Users can browse products, add items to cart, and complete purchases with an intuitive and responsive storefront interface.
- **Role-Based Access Control** - Separate user roles with tailored permissions for regular users and administrators.
- **Admin Dashboard** - Manage products, track orders, and oversee customer activity through a secure administrative interface.
- **MIDTRANS Payment Gateway** - Integration with Midtrans to securely process customer payments.
- **Gmail API Integration** - Automated email notifications for order confirmations, shipping updates, and other customer communications using the Gmail API.
- **Shipping Automation** - Automatically generate professional shipping labels using Puppeteer for PDF rendering and browser automation.
- **Biteship Integration** - Real-time shipping cost estimation powered by Biteship API to ensure accurate checkout pricing.
